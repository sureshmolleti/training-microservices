# Example Logging

## Goal

Logging in unified format.

## Steps

1. `npm install` and `npm start`
2. Open http://localhost:3001
3. Check out `stdout`
4. Restart app with `LOG_LEVEL=debug npm start`
5. Open http://localhost:3001
6. Check out `stdout`
